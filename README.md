# devops-from-zero
#### Create all these files
`touch Makefile`

`touch requirements.txt`

`mkdir devopslib`

`touch devopslib/__init__.py`

`touch hello.py`

`touch test_hello.py`

#### Create virtual environment

1. Create a virtual env 
`virtualenv ~/.venv`

2. Source the venv
Edit .bashrc 
`source ~/.venv/bin/activate`
